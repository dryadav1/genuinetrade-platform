# GenuineTrade Workflow

## 1. Exporter Onboarding
- Exporter signs up
- Submits IEC, GST, company details
- G-Check verification (manual/AI)

## 2. Buyer Onboarding
- Buyer signs up
- Submits business credentials
- Verified before access

## 3. Product Listing
- Exporter adds product (HSN-based)
- Product visible to verified buyers

## 4. RFQ Flow
- Buyer sends RFQ
- Exporter receives and responds

## 5. Communication
- Chat / Email / Docs exchange

## 6. Deal Closure
- Agreement + Invoice
- Payment (future: escrow)

## 7. Trust Update
- G-Score updated based on behavior
